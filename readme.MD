# Customer Analysis using MySQL

## Project Overview
This project analyzes customer data using MySQL. 
The goal is to explore customer distribution, data quality, and subscription activity.

## Dataset
- customers-100.csv
- 100 customer records
- Fields include: Customer ID, Name, Company, Country, Email, Subscription Date

## Analysis Performed
1. Count customers by country
2. Count customers by company
3. Identify newest 10 customers
4. Detect missing email records

## Key Insights
- Countries with highest customer concentration
- Most frequent companies
- Number of missing emails
- Recent subscription trends

## Tools Used
- MySQL Workbench
- SQL (GROUP BY, ORDER BY, COUNT, WHERE)
- CSV export for reporting

## Project Structure
- data/ → raw dataset
- analysis/ → query outputs
- README.md → project documentation
